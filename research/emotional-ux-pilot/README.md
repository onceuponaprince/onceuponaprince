# Emotional UX in LLMs — pilot

**Status: pilot. Episode 1 of a potential spinoff series from the Command Centre campaign. Not a commitment to a full series.**

## What this is

A lateral research thread surfacing from Scene 05's post-conclusion note (2026-04-21). The claim under test: the off-topic asides, hedges, and insight footers that appear in current-generation LLM outputs are load-bearing UX primitives, not stylistic decoration. The pilot makes the premise explicit and tests whether the question is interesting enough to return to.

Framed as a pilot for a spinoff series in the sense of television: self-contained enough to stand alone, hinting at a wider premise worth developing. If it lands, Episode 2 is an ablation study in a real product. If it does not, it stands alone as a well-formed question that happened to fall short.

## How to read it

Open `pilot.html` in a browser. It is a single self-contained file: hybrid essay plus three interactive elements. No build step, no external dependencies.

- Warm vs cold comparison viewer (four example pairs)
- Mechanism spotter (hover the coloured phrases in the sample response)
- Warmth slider thought experiment (projects a hypothetical session-length curve)
